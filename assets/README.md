# Assets Directory

This directory should contain the following image assets:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

You can create placeholder images or use actual assets for your app. The app will work without these, but they are recommended for production.
